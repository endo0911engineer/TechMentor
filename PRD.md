# PRD.md

日本ITエンジニア給与データベース

---

# 1. サービス概要

本プロジェクトは、日本のIT企業におけるソフトウェアエンジニアの給与情報、面接情報、キャリア情報を集約するWebプラットフォームを開発するものである。

ユーザーは以下の情報を閲覧できる。

・企業ごとのエンジニア年収
・面接プロセス
・企業の技術スタック
・働き方（リモート可否など）
・ユーザー投稿による体験談

また、ユーザーは匿名で給与情報や面接体験を投稿できる。

本サービスは、海外サービスである
Levels.fyi に近いコンセプトを、日本のIT企業市場向けに提供する。

目的は、日本のエンジニア転職市場における情報の非対称性を解消することである。

# 3. サービスコンセプト

各IT企業ごとに「企業ページ」を作成し、以下の情報を集約する。

1. 企業概要
2. エンジニア年収データ
3. 面接プロセス
4. ユーザー投稿（体験談）

サイト構造としては以下の要素を組み合わせる。

・企業データベース
・ユーザー投稿データ
・SEO記事

---

# 4. MVPの範囲

MVPでは以下の機能を実装する。

・企業データベース
・企業詳細ページ
・給与投稿機能
・体験談投稿機能
・投稿データの集計表示
・SEO記事ページ

以下の機能はMVPでは実装しない。

・有料プラン
・企業採用掲載
・転職マッチング
・メッセージ機能

まずはデータとトラフィックの獲得を優先する。

・設立年
